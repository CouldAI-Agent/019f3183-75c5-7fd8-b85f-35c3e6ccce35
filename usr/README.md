# AI Med Assistant

Инновационное приложение для здравоохранения, объединяющее пациентов и врачей с помощью технологий искусственного интеллекта.

## Описание

Приложение AI Med Assistant решает две ключевые задачи:
1. **Для врачей:** Автоматизация приема. Во время разговора с пациентом приложение записывает аудио, а ИИ автоматически формирует карту приема, анализирует жалобы и предлагает возможные диагнозы, экономя время врача.
2. **Для пациентов:** Самодиагностика с помощью ИИ, основанная на симптомах и результатах анализов. Если ответа ИИ недостаточно, пациент может задать вопрос реальным врачам (как платно, так и бесплатно).

## Основной функционал

### Для пациентов (Подписка: $10/мес)
*   **ИИ-диагностика:** Анализ симптомов и загруженных анализов.
*   **Консультации с врачами:**
    *   Платный вопрос ($6 - $8) с гарантированным ответом.
    *   Бесплатный вопрос в общую ленту.
*   **Отчеты по здоровью:** Интеграция с умными часами для еженедельной аналитики.
*   **Отзывы и рейтинги:** Выбор лучших специалистов на основе оценок других пациентов.

### Для врачей (Подписка: $25/мес)
*   **ИИ-ассистент на приеме:** Запись разговора и автоматическое заполнение карты.
*   **Ответы на вопросы:** Возможность отвечать на платные вопросы пациентов и зарабатывать.
*   **Верификация:** Строгая система проверки документов и сертификатов.
*   **Рейтинговая система:** Формирование репутации на платформе.

## Разработка

Приложение построено на Flutter и поддерживает работу на мобильных устройствах, планшетах, десктопе и в веб-браузере.

### Запуск проекта

1. Убедитесь, что у вас установлен [Flutter](https://flutter.dev/docs/get-started/install).
2. Клонируйте репозиторий.
3. Выполните команду `flutter pub get` для установки зависимостей.
4. Запустите приложение с помощью `flutter run`.
